## How to Use This Folder

1. **Start with** `01_pm_fundamentals` to refresh core PM knowledge
2. **Read** `02_automotive_technical_knowledge` — interviewers in automotive expect you to know ASPICE, ISO 26262, V-Model
3. **Practise** `03_star_scenarios` out loud — most automotive PM interviews are competency-based
4. **Review** `04` and `05` for situation-specific questions
5. **Memorise** key metrics from `06` — budget variance, SPI, CPI are commonly asked
6. **Drill** `07_interview_200_qa` for rapid-fire Q&A rounds

## Key Automotive PM Differentiators vs Generic PM

| Aspect | Generic PM | Automotive PM |
|--------|-----------|---------------|
| Process standard | PMBOK / Prince2 | **ASPICE (Automotive SPICE)** |
| Safety standard | General risk mgmt | **ISO 26262 (Functional Safety)** |
| SW architecture | N/A | **AUTOSAR, SOVD, UDS** |
| Development model | Waterfall / Agile | **V-Model + Agile hybrid** |
| Supplier mgmt | Procurement | **Tier-1 / Tier-2 / OEM structure** |
| Release gating | Go/No-Go | **SOP (Start of Production) gates** |
| Compliance | ISO 9001 | **IATF 16949, ISO 26262, UNECE R155** |
