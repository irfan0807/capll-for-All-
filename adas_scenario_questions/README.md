# ADAS Scenario-Based Interview Questions

## Overview

This folder contains **100 scenario-based interview questions** on Advanced Driver Assistance Systems (ADAS), organized by feature domain. Each question includes:

- A realistic test/engineering scenario
- Detailed explanation of expected behavior
- Edge cases and failure modes
- Validation approach and acceptance criteria
- Related standards and regulations (ISO 26262, UN ECE R152, NCAP, etc.)

## Key Standards Referenced

- **ISO 26262** – Functional Safety for Road Vehicles
- **ISO 21448 (SOTIF)** – Safety of the Intended Functionality
- **UN ECE R152** – AEB regulation for passenger cars
- **Euro NCAP** – New Car Assessment Programme protocols
- **AUTOSAR** – Automotive Open System Architecture
- **ISO/SAE 21434** – Cybersecurity Engineering
- **SAE J3016** – Levels of Driving Automation (L0–L5)
